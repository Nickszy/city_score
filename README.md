# city_score

从中经网获取数据，给城市的不同方面进行打分

## 文件结构

```
  |----原始下载数据  // 数据来源于中经网
  |----长三角城市年度数据  //对同一个市的数据进行组合
  |----notebook  // ELT
  |    |----EDA   // 探索
  |    |----score  // 结果程序
  |----image   //计算输出的图片
  |----细分指标  // 过程结果
  |----指标.xlsx  // 最终结果
  |----README.md
```


## 教育

### 细分指标

1. 教育投入占比 = 当年GDP / 一般公共预算支出_教育_市辖区
2. 师生比 = 所有学生 / 所有老师
3. 公共图书馆占比 = '公共图书馆总藏量_市辖区' /  '城市户籍人口数'

### 计算

1. 数据归一化((data-mean)/(max-min))
2. 将指标相加(教育投入占比+师生比+公共图书馆占比)
3. round(data+0.5)*10
